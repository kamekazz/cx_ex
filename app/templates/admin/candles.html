About Candle Data

This system automatically aggregates price tick data into OHLC (Open, High, Low, Close) candlestick data for charting.

Data is collected at 1-hour, 4-hour, and daily intervals. The candle aggregator runs every 5 minutes to generate new candles.

API Access

Access candle data programmatically via the API:

GET /admin/api/candles/{stock_id}?interval={1h|4h|1d}&limit={number}

Example: /admin/api/candles/1?interval=1h&limit=50
